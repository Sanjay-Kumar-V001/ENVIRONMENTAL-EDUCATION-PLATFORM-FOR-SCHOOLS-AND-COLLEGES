# EcoLearn: Environmental Education Platform for Schools and Colleges

EcoLearn is a simple beginner-friendly web project that helps students learn environmental topics through lessons, quizzes, and eco-challenges.

## Features
- Home page with introduction.
- Lessons page with environmental topics.
- Interactive quiz with scoring.
- Eco-challenges page with tasks.
- Dashboard page to show points.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the project files.
2. Open `index.html` in a browser.
3. Navigate through the pages using the menu.

1. Upload the project files to a GitHub repository.
2. Go to repository settings.
3. Enable GitHub Pages from the main branch.
4. Open the generated live link.

## Project Goal
The goal of this project is to spread awareness about environmental protection and encourage students to adopt eco-friendly habits.
